Настройка Proxmox для Ansible:
Создайте учетную запись Ansible в Proxmox с необходимыми разрешениями:
Войдите в веб-интерфейс Proxmox
Перейдите в "Permissions" -> "Roles" и создайте новую роль, например, "ansible"
Назначьте роль "ansible" пользователю, который будет использоваться Ansible (например, "ansible")

Устанавливает необходимые пакеты, включая Docker и Docker Compose.
Создает Docker-сеть с именем proxmox_network.
Создает и настраивает контейнеры для каждого сервиса:
qBittorrent
Home Assistant
Kerberos.io
Immich
Scrutiny
Убедитесь, что вы настроили правильные пути для хранения данных (/mnt/data/...) и что версии сервисов соответствуют вашим требованиям.
