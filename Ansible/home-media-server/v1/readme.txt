CSS стили:

Добавлены стили для улучшения внешнего вида страницы.
Используется простой и приятный стиль с центровкой содержимого, изменением цвета фона и кнопками с закругленными углами.
JavaScript для виджета погоды:

Добавлен JavaScript код, который делает запрос к API погоды (OpenWeatherMap) для получения текущих данных о погоде в Москве.
Обновите YOUR_API_KEY на ваш API ключ от OpenWeatherMap.
Шаги для выполнения:
Создайте и отредактируйте playbook, как показано выше.
Убедитесь, что ваш inventory файл настроен правильно и Ansible может подключаться к целевым хостам.
Получите API ключ от OpenWeatherMap и замените YOUR_API_KEY в JavaScript коде на ваш ключ.
