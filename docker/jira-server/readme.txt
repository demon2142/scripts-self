Сервис jira:
Использует официальный Docker-образ Atlassian Jira Software последней версии.
Публикует порт 8080 контейнера на порт 8080 хоста.
Задает необходимые переменные окружения для Jira:
JIRA_HOME - путь к домашнему каталогу Jira.
CATALINA_OPTS - параметры JVM для Jira.
JDBC_URL, JDBC_USER, JDBC_PASSWORD - параметры подключения к базе данных.
Монтирует том jira-home для хранения данных Jira.
Зависит от сервиса db (база данных).
Сервис db:
Использует образ PostgreSQL версии 12.
Задает необходимые переменные окружения для базы данных:
POSTGRES_DB - имя базы данных.
POSTGRES_USER, POSTGRES_PASSWORD - учетные данные пользователя.
Монтирует том jira-db для хранения данных базы данных.
Тома:
jira-home - для хранения данных Jira.
jira-db - для хранения данных базы данных.
